# Я УСТАЛ ДЕЛАТЬ ГРЁБАНЫЕ ПРЕЗЕНТАЦИИ

Чтобы этот экспириенс был чуть лучше я решил написать скриптик, который бы убирал водяной знак Gamma AI из нагенеренных мною през.
Тут резко появилась лаба со свободным условием... Ну вот так и готово. Чтобы показать что я умею использовать дбшку накинул пресеты (текст, который будет подставляться вместо водяного знака) и рейтинг по тому, сколько каждый пользователь очистил нагенеренных слайдов (нормальную систему логина было делать лень).

## КОРСЫ
Шпионская страничка на 127.0.0.1:8080/spy может доставать данные с localhost:8080
